# dahliaOS dotfiles
Dotfiles for creating a daily-drivable Hyprland setup that resembles **[dahliaOS](https://dahliaos.io)** in its looks.

## Default Applications
- Shell: **Bash**
- Terminal: **Alacritty**
- File manager: **PCManFM-Qt**
- Archiver: **LXQt Archiver**

## Dependencies
- parallel
- jaq
- socat
- gtk+3
- gnome-themes-extra
- gtk-engine-murrine
- sassc
